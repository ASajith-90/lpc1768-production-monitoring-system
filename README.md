# lpc1768-production-monitoring-system
An LPC1768 based Industrial Production Monitoring System that monitors product count, defect count, production rate and efficiency using external interrupts, timer, UART and LCD.

# Industrial Production Monitoring System

## Overview
This project is developed using LPC1768 ARM Cortex-M3 microcontroller.

## Features
- Product Counting
- Defect Detection
- Production Rate Calculation
- Efficiency Monitoring
- UART Communication
- LCD Display
- Maintenance Alert using LEDs and Buzzer

## Components Used
- LPC1768
- 16x2 LCD
- LEDs
- Buzzer
- Sensors

## Software Used
- Embedded C
- Keil uVision
